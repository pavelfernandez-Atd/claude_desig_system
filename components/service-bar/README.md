# Service Bar

Page-level header card combining three stacked rows: identity (icon + title + date picker), tab navigation (pill rail with Tab Items), and optional filter bar.

## Quick Start

```html
<link rel="stylesheet" href="service-bar.css">
```

See `demo.html` for a fully working example.

| Class | Purpose |

| `.service-bar` | Card container |

| `.service-bar__text` | Title + subtitle column |
| `.service-bar__title` | Primary heading |

| `.service-bar__filter-btn` | Filter action button |
