# Button

Versatile action button with multiple variants: primary, secondary, plain, outlined, red, green, plain-white, plain-dark. Available in standard (40px), small (30px), and icon-only sizes.

## Quick Start

```html
<link rel="stylesheet" href="button.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.btn` | Base button — 40px height |
| `.btn--sm` | Small variant — 30px height |
| `.btn--accent` | Accent height — 36px |
| `.btn--primary` | Blue filled primary |
| `.btn--secondary` | Light blue secondary |
| `.btn--plain` | Transparent with blue text |
| `.btn--outlined` | White with blue border |
| `.btn--red` | Red destructive |
| `.btn--green` | Green success |
| `.btn--plain-white` | Transparent with white text |
| `.btn--plain-dark` | Transparent with dark text |
| `.btn--icon-only` | Square icon-only button |
| `.btn--icon-primary` | Blue filled icon button |
| `.btn--icon-secondary` | Light blue icon button |
| `.btn--icon-outlined` | Outlined icon button |
| `.btn--disabled` | Disabled state |
| `.btn__icon` | 16×16 icon wrapper |
