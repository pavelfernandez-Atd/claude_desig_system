# Checkbox

Custom styled checkbox with default, hover, checked, and disabled states. Uses appearance:none with a CSS background-image checkmark.

## Quick Start

```html
<link rel="stylesheet" href="checkbox.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.ds-checkbox-input` | Custom checkbox input |
| `.ds-checkbox-label` | Label wrapper |
| `.ds-checkbox-label--disabled` | Disabled label style |


## Design Tokens

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
