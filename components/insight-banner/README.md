# Insight Banner

Informational inline banner with a circular icon wrap, an "INSIGHT" label eyebrow, and a body text paragraph. Used to surface AI-generated insights within a card.

## Quick Start

```html
<link rel="stylesheet" href="insight-banner.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.insight-banner` | Blue-tinted banner container |
| `.insight-banner__icon-wrap` | Circular icon container |
| `.insight-banner__text` | Text column |
| `.insight-banner__label` | Uppercase eyebrow label |
| `.insight-banner__body` | Body paragraph |
