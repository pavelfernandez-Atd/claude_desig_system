# Action Banner

Recommendation inline banner with a green-tinted icon wrap, an "ACTION" label eyebrow, and a body that supports paragraphs and bullet lists.

## Quick Start

```html
<link rel="stylesheet" href="action-banner.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.action-banner` | Green-tinted banner container |
| `.action-banner__icon-wrap` | Circular icon container |
| `.action-banner__text` | Text column |
| `.action-banner__label` | Uppercase eyebrow label |
| `.action-banner__body` | Body (supports p + ul) |


## Design Tokens

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
