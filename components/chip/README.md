# Chip

Compact label badge with outlined, light, and dark style variants across 14 semantic colours. Supports an optional 16×16 icon and text label.

## Quick Start

```html
<link rel="stylesheet" href="chip.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.ds-chip` | Base chip — 26px height |
| `.ds-chip__icon` | 16×16 icon slot |
| `.ds-chip__label` | Text label |
| `.ds-chip--outlined` | Outlined style variant |
| `.ds-chip--light` | Light/pastel style variant |
| `.ds-chip--dark` | Dark/filled style variant |
| `.ds-chip--blue/green/red/…` | Colour modifier (14 options) |


## Design Tokens

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
