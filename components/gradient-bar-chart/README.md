# Gradient Bar Chart

Horizontal bar chart where each bar is a gradient fill anchored at the left. Right-aligned label column, bar tracks, inline value labels and outside value labels. Hover shows a tooltip with additional detail.

## Quick Start

```html
<link rel="stylesheet" href="gradient-bar-chart.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.gbc` | Root chart container |
| `.gbc__row` | Single bar row (label + track) |
| `.gbc__label` | Right-aligned text label |
| `.gbc__track` | Bar track (full width flex) |
| `.gbc__fill` | Gradient filled bar |
| `.gbc__val` | Value inside bar |
| `.gbc__val-outside` | Value outside bar (small values) |
| `.gbc__tooltip` | Hover detail tooltip |
| `.gbc__tooltip-text` | Tooltip body text |
