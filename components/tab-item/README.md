# Tab Item

Primary page-level tab pill used in the main tab strip. Supports default, hover, active, and disabled states with optional icon and counter badge.

## Quick Start

```html
<link rel="stylesheet" href="tab-item.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.tab-item` | Base pill, default state |
| `.tab-item--active` | Selected — bright blue fill |
| `.tab-item--disabled` | Disabled — greyed, no pointer events |
| `.tab-item__icon` | 16×16 icon wrapper |
| `.tab-item__counter` | Count badge (flips to white when active) |


## Design Tokens

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
