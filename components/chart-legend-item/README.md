# Chart Legend Item

Compact legend entry combining an optional favicon or colour dot with a text label. Supports a muted/strikethrough off-state for toggling series visibility.

## Quick Start

```html
<link rel="stylesheet" href="chart-legend-item.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.chart-legend-item` | Flex row container |
| `.chart-legend-item__favicon` | 14×14 favicon |
| `.chart-legend-item__dot` | 9×9 colour dot |
| `.chart-legend-item__label` | Label text |
| `.chart-legend-item__label--brand` | Bold brand emphasis |
| `.csd-series--off` | Muted/strikethrough off state |

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
