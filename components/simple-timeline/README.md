# Simple Timeline

SVG-based line chart that scales proportionally with its container width. Dots are clickable and show a fixed tooltip. No JS required for rendering — SVG is pre-built.

## Quick Start

```html
<link rel="stylesheet" href="simple-timeline.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.simple-timeline` | Full-width block container |
| `.simple-timeline__svg` | SVG element (pre-built viewBox) |
| `.simple-timeline__tooltip` | Fixed-position hover tooltip |
| `.simple-timeline__tooltip-series` | Series name in tooltip |
| `.simple-timeline__tooltip-val` | Value in tooltip |
