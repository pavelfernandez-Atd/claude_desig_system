# Sub Tab

Small pill-shaped tab used in dense navigation strips. Supports default, hover, active, and highlighted states, with optional icon and badge count.

## Quick Start

```html
<link rel="stylesheet" href="sub-tab.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.subtab` | Base pill tab, default state |
| `.subtab--active` | Selected — dark fill, white text |
| `.subtab--highlighted` | Outlined variant — white bg with border |
| `.subtab__icon` | 16×16 icon wrapper |
| `.subtab__badge` | Count badge appended after label |


## Design Tokens

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
