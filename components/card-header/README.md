# Card Header

Standard header block for chart and data cards. Features an optional roundel icon, title with optional subtitle, and right-aligned action buttons.

## Quick Start

```html
<link rel="stylesheet" href="card-header.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.card-header` | Header bar — rounded top corners |
| `.card-header--standalone` | All corners rounded (no attached body) |
| `.card-header__roundel` | Circular icon badge |
| `.card-header__title-section` | Title + subtitle column |
| `.card-header__title-row` | Title row (may include chips) |
| `.card-header__title` | Primary heading text |
| `.card-header__subtitle` | Secondary description |
| `.card-header__actions` | Right-aligned action buttons |
| `.card-header__action-btn` | 30×30 icon action button |
