# Stacked Timeline

A 100% stacked area chart for visualising share-of-category breakdowns over time.

## Features

- 100% stacked bands (values always sum to 100%)
- Hover interaction: vertical indicator line + dots at each band boundary + tooltip
- Open Sans Condensed legend
- DS chart palette colours

## Usage

```html
<link rel="stylesheet" href="stacked-timeline.css">
<!-- SVG chart markup — see demo.html -->
```

## Series

| Series | Colour |
|---|---|
| Ads Above AIO | `#8ac8b1` (Green, chart index 05) |
| Ads Inside AIO | `#b3a2e7` (Purple, chart index 01) |
| Ads Below AIO | `#e86492` (Crimson, chart index 04) |
| No Ads | `#c0cde0` (Neutral/50) |
