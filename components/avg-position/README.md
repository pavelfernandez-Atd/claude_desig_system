# Avg Position

Citations average position list showing ranked domains with numeric position badges, favicons, domain labels and delta indicators. Brand rows are highlighted in navy.

## Quick Start

```html
<link rel="stylesheet" href="avg-position.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.cit-pos__list` | Column of position rows |
| `.cit-pos__row` | Single domain row |
| `.cit-pos__num` | Numeric position badge |
| `.cit-pos__num--brand` | Navy brand highlight |
| `.cit-pos__favicon` | Favicon slot |
| `.cit-pos__domain` | Domain label pill |
| `.cit-pos__domain--brand` | Navy brand domain |
| `.cit-pos__delta` | Change indicator text |
| `.cit-pos__delta--up` | Green positive |
| `.cit-pos__delta--down` | Red negative |
| `.cit-pos__delta--flat` | Blue-grey flat |
