# Tooltip Info

Floating information tooltip with a white card body and a centred arrow. Supports arrow-bottom (above trigger) and arrow-top (below trigger) placements.

## Quick Start

```html
<link rel="stylesheet" href="tooltip-info.css">
```

See `demo.html` for a fully working example.

## CSS Classes

| Class | Purpose |
|---|---|
| `.tooltip-info` | Root wrapper — drop-shadow filter |
| `.tooltip-info--arrow-top` | Flips arrow direction (tooltip below trigger) |
| `.tooltip-info__body` | White rounded card with text |
| `.tooltip-info__arrow` | SVG caret shape |


## Design Tokens

All colours and sizing use Adthena Design System CSS custom properties defined in `:root`.
